Journal

Journal Format

Voice and text entries from Voice Inbox will be appended to today’s daily note’s "Inbox" heading:

## Inbox

If the heading or file doesn’t exist, it’ll be created.

Voice and text entries from Voice Inbox will be appended to the ## Inbox heading in today's daily note. If the heading or file does not exist, it will be created.

For entries with only one line, it’s appended as a bullet point with the recording timestamp prefix.

- 17:47: This is a one-line entry

If an entry has multiple lines, it’ll be formatted as a subheading.

### 17:47

This is an entry.
That contains multiple lines.

Here's a full example of a daily journal file.

## Inbox

- 11:00 I got a new idea about ...
- 14:00 The inbox acts as a mental repository, allowing individuals to clear their minds ...

### 15:00

It's time for a checkup.
I just scheduled an appointment with the doctor on next Friday.

Recording

Is there a recording time limit?

The recordings should be relatively short, ideally under 15 minutes. You can record a bunch of entries, but not a super long one. The recording button are not designed for things like recording a whole meeting, and if it’s over 50 minutes, it won’t work for now.",

How to use the built-in speech recognition?

  1. On the Home view.
  2. Tap the pencil at the bottom-right.
  3. When the keyboard is present, tap the microphone at the bottom-right.

Using this is free and fast, but the accuracy is significantly lower.

Calendar Events

How to add calendar events?

Prerequisites:

  • "Add Calendar Events" is enabled.
  • Permission to access the calendar must be granted.
  • A calendar event can only be added via voice entries.

Unlike voice assistants like Siri, there's no need to say:

Add an event to my calendar ...

Instead just mention the event and date will work, for example:

I'll talk to Sam about the project tomorrow at 10 a.m.

These are the details of the event that you might include in the recording:

  • Title
  • All-day
  • StartDate
  • EndDate
  • Location
  • Notes

If there's no event recognized, the Calendar button of the entry on the Inbox view will be greyed out, otherwise, an event will be added to your Calendar. Click the button for details.

Currently, events in the past are ignored.

Can I edit the calendar event?

Not yet.

Editing an event requires "full access" permission to the Calendar at the beginning, it's not something easy to ask for.

If you need this, feel free to drop me an email so I’ll prioritize it. “Hey, I need event editing too.” is fine.
